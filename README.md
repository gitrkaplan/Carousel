# Carousel
A carousel for users who want to display multiple images but one at a time on a time interval.

## Usage
```
$ mongod
$ npm install
$ npm run seed
$ npm run start
```
## Uses
- HTML5
- CSS3
- Javascript(ES2015+)
- Node.js
- Express.js
- MongoDB
